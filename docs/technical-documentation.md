# Technical Documentation

## Technologies
HTML, CSS, JavaScript, GitHub REST API

## Main Features
- Theme toggle with localStorage
- Visitor name storage
- Project details toggle
- Filter and sort logic
- Contact form validation
- GitHub API repository fetch

## State Management
localStorage is used to save:
- theme preference
- visitor name
- selected filter and sort options

## Performance
- responsive layout
- reduced repeated DOM queries
- optimized image usage
