# AI Usage Report

## Tools Used and Use Cases
During this assignment, I used ChatGPT as an AI-assisted tool to support my learning and development process.

I used ChatGPT for:
- generating starter ideas for interactive portfolio features
- improving the structure of my HTML, CSS, and JavaScript files
- suggesting ways to implement dark/light mode using `localStorage`
- helping me add dynamic features such as saving a user name and showing/hiding project details
- suggesting animations and transitions for buttons, cards, links, and section navigation
- helping write and improve project documentation such as the README and technical documentation

## Benefits of Using AI
Using AI helped me:
- save time when planning the project structure
- understand how to connect HTML, CSS, and JavaScript more effectively
- get ideas for features that match the assignment requirements
- improve the user experience of the website with animations and feedback messages
- organize my documentation more clearly

## Challenges and Limitations
Although AI was helpful, there were some challenges:
- some suggestions needed modification to fit my exact portfolio content
- I had to review the generated code carefully to make sure it worked correctly
- not every suggestion matched the assignment immediately, so I needed to adjust parts of the code myself
- I had to test the website and fix formatting or logic issues after using AI suggestions

## Learning Outcomes
From using AI in this assignment, I learned:
- how to use `localStorage` to save theme and user name data
- how to create interactive features such as expanding and collapsing project details
- how to improve user experience with CSS transitions and hover effects
- how to provide feedback messages to users when input is valid or invalid
- how AI can support brainstorming, debugging, and documentation without replacing my own understanding

## Responsible Use and Modifications
I reviewed all AI-generated suggestions before adding them to my project. I edited the code, adapted the text to my own website, and made sure I understood how each part worked. I did not submit raw AI output without checking it.

I tested the features manually in the browser, including:
- dark/light theme switching
- saved theme persistence
- saved name persistence
- project details toggling
- smooth scrolling
- feedback messages for user input

This helped me ensure the final work was correct, original, and aligned with academic integrity requirements.

## Reflection
AI was useful as a support tool during this assignment. It helped me work more efficiently and learn practical front-end development concepts. However, I still needed to understand the code, make changes, and test the final result myself. This made the experience helpful for both completing the assignment and improving my own skills.
